Oompa Loompa Doopity Doo,
If you are wise you already know what to do.

Yeah basically this is a Nexerelin-Compatible Starsector Mod that adds the Oompa Loompas as Pirate Faction into Starsector.
They start off strong in a custom Sector since I wanted to emulate the insane productivity these guys have.
But don't worry, I tried to design them in a way that they absolutely suck at attacking (they are supposed to be a bit of a nuisance but not a menace)
but also have them be difficult to conquer (I always found that, if provoked, they got some crazy ways to just turn the whole situation around)

Do give me feedback if something's fucked!
